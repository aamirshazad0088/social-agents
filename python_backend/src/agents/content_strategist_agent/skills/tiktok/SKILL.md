# TikTok Content Skill

This skill provides structured workflows for creating viral TikTok content.

Use this skill when asked to:
- Create TikTok video scripts
- Develop TikTok content strategies

## Platform Guidelines

### Video Format
- **Length**: 15-60 seconds optimal (can go up to 10 minutes)
- **Aspect ratio**: 9:16 (vertical)
- **Hook timing**: First 1-3 seconds critical
- **Captions**: On-screen text for accessibility

### Content Structure

**Standard Video Script:**
```
[HOOK - 0-3 seconds]
"Here's something that will change how you think about [topic]..."

[CONTEXT - 3-10 seconds]
Set up the problem or situation

[MAIN CONTENT - 10-45 seconds]
Deliver value in digestible chunks

[CTA - Last 5 seconds]
Follow for more / Comment your [X] / Try this and tag me
```

## Hook Templates

### Curiosity Hooks
- "Nobody's talking about this..."
- "I can't believe I just learned this..."
- "Why is no one talking about [topic]?"
- "This changed everything for me..."

### Challenge Hooks
- "Watch til the end..."
- "Can you spot what's different?"
- "90% of people get this wrong..."

### Story Hooks
- "Story time..."
- "The craziest thing just happened..."
- "POV: [situation]"

### Educational Hooks
- "[Number] things I wish I knew about [topic]"
- "Here's your sign to [action]"
- "If you're [situation], you need this..."

## Caption Guidelines

- Keep captions short (under 150 characters for visibility)
- Use 3-5 hashtags
- Include a question or CTA
- Mention trending sounds if applicable

## Hashtag Strategy

### Always Include
- #fyp or #foryou
- #[your niche]
- #[trending topic if relevant]

### Avoid
- Banned hashtags
- Irrelevant trending tags
- Too many hashtags (stick to 3-5)

## Trending Sound Usage

- Check "Discover" page daily
- Use sounds while still trending (first 3-7 days)
- Put your unique spin on trends
- Ensure sound fits your brand

## Quality Checklist

Before finalizing content:
- [ ] Hook captures attention in 0-3 seconds

- [ ] On-screen text included for accessibility
- [ ] CTA is clear and specific
- [ ] Caption includes relevant hashtags
- [ ] Format is 9:16 vertical
